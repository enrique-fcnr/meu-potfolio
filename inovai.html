Sobre o Projeto

O Inovaí é um projeto de site para uma agência de inteligência artificial, que se destaca pela agilidade e automação de serviços. O site foi projetado para ser uma vitrine dos serviços oferecidos, como soluções de IA personalizadas, consultoria e desenvolvimento de sistemas automatizados. Embora o front-end seja um protótipo estático, o foco principal é na arquitetura de back-end e na futura integração com serviços de cloud.

Minha Participação

Neste projeto, minha contribuição foi focada no estudo e planejamento de uma arquitetura de back-end escalável utilizando Java e Spring Boot, com uma visão estratégica para a integração com a AWS. O objetivo era criar uma estrutura robusta que pudesse hospedar os serviços de IA e suportar o crescimento da agência. Meu trabalho incluiu a definição dos modelos de dados, o design da API e a exploração de como os serviços da AWS (como EC2 para hospedagem e S3 para armazenamento de dados) seriam utilizados para otimizar a performance e a segurança do sistema. O projeto serviu como um laboratório prático para aprofundar meus conhecimentos em cloud computing e DevOps, preparando-me para desafios mais complexos na área.
