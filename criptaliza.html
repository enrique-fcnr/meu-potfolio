Sobre o Projeto

O Criptaliza é um sistema de investimento em criptomoedas, desenvolvido com Java e o framework Spring Boot. O objetivo do projeto é oferecer uma plataforma robusta e segura para os usuários gerenciarem seus investimentos em ativos digitais. A aplicação é modular, com uma arquitetura de back-end bem definida para suportar funcionalidades como cotações em tempo real, histórico de transações e um assistente virtual para auxiliar nas decisões de investimento.

Minha Participação

Neste projeto, atuei como desenvolvedor back-end, focando na implementação dos serviços e na lógica de negócio. Utilizei o Spring Boot para criar os endpoints da API REST, gerenciando o fluxo de dados entre o front-end (hipotético) e o banco de dados. Minhas responsabilidades incluíram a modelagem de dados, a implementação de regras de negócio para as transações de criptomoedas e a criação de um sistema de interação com o assistente virtual, que processa e responde às consultas dos usuários. O projeto foi uma excelente oportunidade para aprofundar meu conhecimento em Java e na arquitetura de software, garantindo um código limpo, escalável e de fácil manutenção.
